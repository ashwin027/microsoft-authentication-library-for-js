acquireTokenSilent

  • acquireTokenSilent(scopes: Array<string>, authority?: string, user?: User, extraQueryParameters?: string): Promise<string>
    • Defined in UserAgentApplication.ts:842

    Gets token from the cache if it is not expired. Otherwise sends request to AAD to obtain an access_token using a hidden iframe. To renew idToken, clientId should be passed as the only scope in the scopes array.

    Parameters

    • scopes: Array<string>

      Scopes requested by the user. Scopes like 'openid' and 'profile' are sent with every request.

    • Optional authority: string

      Authority

    • Optional user: User

      The user for which the scopes are requested.The default user is the logged in user.

    • Optional extraQueryParameters: string

      ExtraQueryParameters to add to the authentication request.

    Returns Promise<string>

    Resolved with token or rejected with error.
